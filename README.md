# PockeyQuest

- Auto up lv
- Auto tap
- Support multi account
- Register PockeyQuest [here](https://t.me/pokequest_bot/app?startapp=yvM4U7Npzm)

## Installation
```
git clone https://github.com/himiko3939/Poke
```
```
cd PokeyQuest-BOT
```
```
npm install
```
## Usage
### Edit data.txt
```
nano data.txt
```
### Create screen
```
screen -S your_screen_name
```
### Run
```
node main.js
```
